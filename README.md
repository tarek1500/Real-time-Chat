# Chat
Real-time Chat application based on Laravel & Vue.js .

## Description
Chat application tutorial. In this tutorial we are using Laravel **5.7**, and Vue.js **2.5**.<br>
Here is a list of the packages we have used:

### For server:
- Laravel Websockets.
- Laravel Passport.
### For client:
- Bootstrap 4.
- Laravel Echo.
- Vue.
- Vue Chat Scroll.
- Vue Flash Message.
- Vue Multiselect.
- Vue Resource.
- Vue Router.
- Vuebar.
- Vuex.
